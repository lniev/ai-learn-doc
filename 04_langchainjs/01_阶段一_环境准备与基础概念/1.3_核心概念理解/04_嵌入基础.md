嵌入（Embeddings）学习指南

第一章：什么是嵌入？

一句话理解：嵌入是将文本、图像等数据转换为数字向量的过程，使得语义相似的内容在向量空间中距离相近。

```
"苹果"  →  [0.12, -0.34, 0.89, ..., 0.05]  (1536维向量)
"香蕉"  →  [0.15, -0.29, 0.85, ..., 0.08]  ← 距离很近（都是水果）
"汽车"  →  [-0.76, 0.42, -0.12, ..., 0.91]  ← 距离很远（不同语义）
```

为什么需要嵌入？

- 机器只能理解数字，不能理解文字
- 通过向量相似度计算语义相关性
- 支持大规模语义搜索（在海量文档中找到最相关的内容）

---

第二章：第一个嵌入程序

使用 OpenAI 的嵌入模型（最常用）：

```javascript
import { OpenAIEmbeddings } from "@langchain/openai";

// 1. 创建嵌入模型实例
const embeddings = new OpenAIEmbeddings({
	modelName: "text-embedding-3-small", // 推荐：便宜且效果好
	// modelName: "text-embedding-3-large", // 高质量但贵
	// modelName: "text-embedding-ada-002", // 旧版，兼容性好
});

// 2. 嵌入单个文本
const singleVector = await embeddings.embedQuery("这是一段测试文本");
console.log(singleVector.length); // 1536（向量维度）
console.log(singleVector.slice(0, 5)); // 查看前5个数字

// 3. 嵌入多个文档（批量，更省成本）
const documents = [
	"机器学习是人工智能的一个分支",
	"深度学习是机器学习的子集",
	"Python 是一种编程语言",
];
const vectors = await embeddings.embedDocuments(documents);
console.log(vectors.length); // 3（返回3个向量）
```

第三章：理解向量相似度

嵌入的核心应用是比较向量之间的相似度：

```javascript
import { OpenAIEmbeddings } from "@langchain/openai";
import { cosineSimilarity } from "@langchain/core/utils/math"; // LangChain 内置工具

const embeddings = new OpenAIEmbeddings();

// 准备查询和候选文档
const query = "我想学习人工智能";
const candidates = [
	"机器学习课程介绍",
	"Python 入门教程",
	"深度学习实战指南",
	"如何制作蛋糕",
];

// 生成向量
const queryVector = await embeddings.embedQuery(query);
const candidateVectors = await embeddings.embedDocuments(candidates);

// 计算相似度并排序
const similarities = candidateVectors.map((vec, idx) => ({
	text: candidates[idx],
	score: cosineSimilarity(queryVector, vec),
}));

similarities.sort((a, b) => b.score - a.score);

console.log("查询：", query);
console.log("最相关的结果：");
similarities.forEach(item => {
	console.log(`  [${item.score.toFixed(4)}] ${item.text}`);
});
// 输出：
// [0.8234] 深度学习实战指南
// [0.7891] 机器学习课程介绍
// [0.2341] Python 入门教程
// [0.0567] 如何制作蛋糕
```

相似度计算方法：

- 余弦相似度（最常用）：-1 到 1，1 表示完全相同
- 欧氏距离：越小越相似
- 点积：考虑向量长度，适合同一模型生成的向量
