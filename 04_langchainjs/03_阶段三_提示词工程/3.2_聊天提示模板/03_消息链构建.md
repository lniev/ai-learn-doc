3.2 聊天提示模板 + 03 消息链构建

核心概念

概念 说明
聊天提示模板 构建结构化对话的模板系统
消息链 按顺序连接多个消息模板，形成完整对话流

```
消息链结构：
SystemMessage  → 设定全局规则
    ↓
HumanMessage   → 用户输入（可多个示例）
    ↓
AIMessage      → AI回复（少样本示例）
    ↓
HumanMessage   → 当前真实输入
```

基础消息链构建

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";

// 方式1：数组简写（推荐）
const prompt1 = ChatPromptTemplate.fromMessages([
	["system", "你是{role}，擅长{skill}。"],
	["human", "你好"],
	["ai", "您好！我是您的{role}，有什么可以帮您？"],
	["human", "{question}"],
]);

// 方式2：显式消息对象（了解原理）
import {
	SystemMessagePromptTemplate,
	HumanMessagePromptTemplate,
	AIMessagePromptTemplate,
} from "@langchain/core/prompts";

const prompt2 = ChatPromptTemplate.fromMessages([
	SystemMessagePromptTemplate.fromTemplate("你是{role}，擅长{skill}。"),
	HumanMessagePromptTemplate.fromTemplate("你好"),
	AIMessagePromptTemplate.fromTemplate(
		"您好！我是您的{role}，有什么可以帮您？",
	),
	HumanMessagePromptTemplate.fromTemplate("{question}"),
]);
```

动态消息链构建

根据条件添加消息

```javascript
function buildDynamicPrompt(options) {
	const messages = [["system", "你是{role}。"]];

	// 条件添加少样本示例
	if (options.examples && options.examples.length > 0) {
		for (const ex of options.examples) {
			messages.push(["human", ex.question]);
			messages.push(["ai", ex.answer]);
		}
	}

	// 条件添加上下文
	if (options.context) {
		messages.push(["system", "背景信息：{context}"]);
	}

	// 添加当前输入
	messages.push(["human", "{input}"]);

	return ChatPromptTemplate.fromMessages(messages);
}

// 使用
const prompt = buildDynamicPrompt({
	examples: [
		{ question: "Q1", answer: "A1" },
		{ question: "Q2", answer: "A2" },
	],
	context: "用户是初学者",
});

const result = await prompt.formatMessages({
	role: "导师",
	context: "用户是初学者",
	input: "我的问题",
});
```

消息链与 LCEL 结合

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";
import {
	RunnableSequence,
	RunnablePassthrough,
} from "@langchain/core/runnables";

// 构建复杂处理链
const chain = RunnableSequence.from([
	// 步骤1：注入动态上下文
	RunnablePassthrough.assign({
		timestamp: () => new Date().toISOString(),
		historySummary: input => summarizeHistory(input.history),
	}),

	// 步骤2：构建消息链
	input => {
		const prompt = ChatPromptTemplate.fromMessages([
			["system", "你是{role}。当前时间：{timestamp}"],
			["system", "对话摘要：{historySummary}"],
			...input.recentMessages.flatMap(m => [[m.role, m.content]]),
			["human", "{currentInput}"],
		]);

		return prompt.formatMessages(input);
	},

	// 步骤3：调用模型
	model,
]);
```

消息链的最佳实践

```javascript
// 好的消息链：结构清晰，角色明确
const goodPrompt = ChatPromptTemplate.fromMessages([
	// 1. 系统设定（全局规则）
	["system", "你是{role}。约束：{constraints}"],

	// 2. 少样本示例（展示期望行为）
	["human", "示例输入1"],
	["ai", "示例输出1（符合约束）"],

	// 3. 动态上下文（可选）
	["system", "当前上下文：{context}"],

	// 4. 历史对话（最近N轮）
	["human", "{history[-2].input}"],
	["ai", "{history[-2].output}"],
	["human", "{history[-1].input}"],
	["ai", "{history[-1].output}"],

	// 5. 当前输入
	["human", "{input}"],
]);

// 差的消息链：结构混乱，信息缺失
const badPrompt = ChatPromptTemplate.fromMessages([
	["human", "你是{role}，帮我{task}"], // ❌ 角色应放system
	["ai", "好的"], // ❌ 无实质内容
	["human", "{input}"], // ❌ 缺少上下文
]);
```
