# 提示词模板（Prompt Template）详解

## 核心概念

```
Prompt Template = 预定义模板 + 动态变量

作用：将用户输入结构化，转化为模型可理解的指令格式
```

---

## 两种模板类型

| 类型                   | 用途                   | 输出格式           |
| ---------------------- | ---------------------- | ------------------ |
| **PromptTemplate**     | 基础文本补全（LLM）    | 字符串 String      |
| **ChatPromptTemplate** | 对话模型（Chat Model） | 消息数组 Message[] |

---

## 基础模板（PromptTemplate）

### 基本用法

```javascript
import { PromptTemplate } from "@langchain/core/prompts";

// 方式1：fromTemplate 静态方法（推荐）
const template = PromptTemplate.fromTemplate(
	"将以下{language}代码翻译成Python：\n\n{code}",
);

// 方式2：构造函数
const template = new PromptTemplate({
	template: "将以下{language}代码翻译成Python：\n\n{code}",
	inputVariables: ["language", "code"],
});
```

### 变量注入与格式化

```javascript
// 格式化：传入变量值
const formatted = await template.format({
	language: "JavaScript",
	code: "const x = 10;",
});

// 输出结果：
// 将以下JavaScript代码翻译成Python：
//
// const x = 10;
```

## 聊天模板（ChatPromptTemplate）

### 多角色消息结构

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";

const template = ChatPromptTemplate.fromMessages([
	// 系统消息：设定角色和行为
	["system", "你是一位资深{role}，擅长{skill}。"],

	// 人类消息：用户输入
	["human", "请帮我{task}：\n{content}"],

	// AI消息（可选）：示例回复，用于少样本学习
	["ai", "好的，我会..."],
]);
```

### 实际调用

```javascript
const messages = await template.formatMessages({
	role: "Python工程师",
	skill: "代码优化",
	task: "重构这段代码",
	content: "def foo(): pass",
});

// 输出：
// [
//   SystemMessage { content: "你是一位资深Python工程师..." },
//   HumanMessage { content: "请帮我重构这段代码：\ndef foo(): pass" }
// ]
```

## 高级模板技巧

### 技巧1：部分预设（Partial）

```javascript
// 预设部分变量，创建新模板
const codeTemplate = PromptTemplate.fromTemplate(
	"语言：{language}\n代码：{code}\n解释：",
);

// 固定语言为Python
const pythonTemplate = await codeTemplate.partial({
	language: "Python",
});

// 后续只需传入 code
const result = await pythonTemplate.format({
	code: "print('hello')",
});
```

### 技巧2：函数动态填充

```javascript
import { PromptTemplate } from "@langchain/core/prompts";

const template = new PromptTemplate({
	template: "今天是{date}，{greeting}",
	inputVariables: ["greeting"],
	partialVariables: {
		date: () => new Date().toLocaleDateString(), // 动态获取
	},
});
```

### 技巧3：管道组合（LCEL）

```javascript
import { PromptTemplate } from "@langchain/core/prompts";
import { ChatOpenAI } from "@langchain/openai";

const template = PromptTemplate.fromTemplate("讲一个关于{topic}的笑话");
const model = new ChatOpenAI({ temperature: 0.9 });

// 管道：模板 → 模型 → 输出
const chain = template.pipe(model);

const result = await chain.invoke({ topic: "程序员" });
```

## 少样本模板（Few-Shot）

### 带示例的模板

```javascript
import { FewShotPromptTemplate, PromptTemplate } from "@langchain/core/prompts";

// 定义示例
const examples = [
	{ input: "开心", output: "positive" },
	{ input: "难过", output: "negative" },
	{ input: "还行", output: "neutral" },
];

// 示例格式化模板
const examplePrompt = PromptTemplate.fromTemplate(
	"输入：{input}\n输出：{output}\n",
);

// 组装少样本模板
const fewShotPrompt = new FewShotPromptTemplate({
	examples: examples,
	examplePrompt: examplePrompt,
	prefix: "对以下评论进行情感分类：",
	suffix: "输入：{input}\n输出：",
	inputVariables: ["input"],
});

const result = await fewShotPrompt.format({
	input: "这个产品太棒了！",
});

// 输出包含3个示例 + 新问题
```

## 快速选择指南

| 场景              | 模板类型                | 代码                                     |
| ----------------- | ----------------------- | ---------------------------------------- |
| 简单文本生成      | `PromptTemplate`        | `PromptTemplate.fromTemplate("...")`     |
| 多轮对话/角色扮演 | `ChatPromptTemplate`    | `ChatPromptTemplate.fromMessages([...])` |
| 需要示例学习      | `FewShotPromptTemplate` | `new FewShotPromptTemplate({...})`       |
| 动态变量预设      | `partial()`             | `template.partial({var: value})`         |
| 链式组合          | LCEL管道                | `template.pipe(model).pipe(parser)`      |

---

## 最佳实践

| 实践             | 说明                            |
| ---------------- | ------------------------------- |
| **变量命名清晰** | `{customer_name}` 优于 `{name}` |
| **添加格式说明** | 用 `\n\n` 分隔不同部分          |
| **预留思考空间** | 复杂任务加"请逐步思考"          |
| **验证模板输出** | `console.log` 检查格式化结果    |
| **复用模板实例** | 避免重复创建，提升性能          |

---

## 一句话总结

```
PromptTemplate = 填空题试卷（LLM用）
ChatPromptTemplate = 角色扮演剧本（ChatModel用）

核心操作：定义模板 → 注入变量 → format格式化 → 送入模型
```
