03 模板格式化

核心概念

模板格式化 = 预定义文本结构 + 动态变量注入，实现提示词的标准化和复用。

```
模板: "你好，{name}！今天是{day}。"
      ↓ 注入 {name: "张三", day: "周一"}
结果: "你好，张三！今天是周一。"
```

---

基础模板（PromptTemplate）

```javascript
import { PromptTemplate } from "@langchain/core/prompts";

// 1. 创建模板
const template = PromptTemplate.fromTemplate(
  "请用一句话介绍{topic}，要求{style}风格。"
);

// 2. 格式化（注入变量）
const formatted = await template.format({
  topic: "区块链",
  style: "幽默"
});

console.log(formatted);
// 输出: 请用一句话介绍区块链，要求幽默风格。
```

---

模板 + 模型调用

```javascript
import { PromptTemplate } from "@langchain/core/prompts";
import { ChatOpenAI } from "@langchain/openai";

const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });

// 创建模板
const template = PromptTemplate.fromTemplate(
  "将以下{text}翻译成{language}："
);

// 方式1：先格式化，再调用
const prompt = await template.format({
  text: "Hello World",
  language: "中文"
});
const result1 = await model.invoke(prompt);

// 方式2：用 LCEL 链式（推荐）
const chain = template.pipe(model);
const result2 = await chain.invoke({
  text: "Hello World",
  language: "中文"
});
```

聊天模板（ChatPromptTemplate）

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";

// 多角色模板
const chatTemplate = ChatPromptTemplate.fromMessages([
  ["system", "你是{role}，擅长{skill}。"],
  ["human", "请帮我{task}"]
]);

// 格式化
const messages = await chatTemplate.formatMessages({
  role: "前端专家",
  skill: "React",
  task: "优化性能"
});

console.log(messages);
// [
//   SystemMessage { content: "你是前端专家，擅长React。" },
//   HumanMessage { content: "请帮我优化性能" }
// ]

// 直接调用
const chain = chatTemplate.pipe(model);
const result = await chain.invoke({
  role: "前端专家",
  skill: "React",
  task: "优化性能"
});
```

模板变量高级用法

默认值

```javascript
const template = new PromptTemplate({
  template: "你好{name}，今天是{day|周一}。",  // | 后默认值（部分版本支持）
  inputVariables: ["name", "day"],
  partialVariables: { day: "周一" }  // 编程方式设置默认值
});

// 或运行时 partial
const withDay = await template.partial({ day: "周五" });
const formatted = await withDay.format({ name: "张三" });
```

条件判断（简单逻辑）

```javascript
import { PromptTemplate } from "@langchain/core/prompts";

// 使用函数预处理变量
const template = PromptTemplate.fromTemplate(`
  {greeting}，{name}！
  {statusMsg}
`);

const vars = {
  name: "张三",
  isVip: true,
  greeting: "欢迎",
  statusMsg: (vars) => vars.isVip ? "您是尊贵的VIP用户。" : "普通用户您好。"
};

// 预处理函数变量
const processedVars = {
  ...vars,
  statusMsg: typeof vars.statusMsg === 'function'
    ? vars.statusMsg(vars)
    : vars.statusMsg
};

const result = await template.format(processedVars);
// 输出: 欢迎，张三！您是尊贵的VIP用户。
```

少样本模板（Few-Shot）

```javascript
import { PromptTemplate, FewShotPromptTemplate } from "@langchain/core/prompts";

// 定义示例
const examples = [
  { input: "高兴", output: "positive" },
  { input: "难过", output: "negative" },
  { input: "平静", output: "neutral" }
];

// 单个示例的模板
const examplePrompt = PromptTemplate.fromTemplate(`
  文本: {input}
  情感: {output}
`);

// 组合少样本模板
const fewShotPrompt = new FewShotPromptTemplate({
  examples,           // 示例数组
  examplePrompt,      // 单个示例格式
  prefix: "判断以下文本的情感（positive/negative/neutral）：",
  suffix: "文本: {input}\n情感:",
  inputVariables: ["input"],
  exampleSeparator: "\n---\n"
});

const formatted = await fewShotPrompt.format({ input: "兴奋" });
console.log(formatted);
// 判断以下文本的情感（positive/negative/neutral）：
//
// 文本: 高兴
// 情感: positive
// ---
// 文本: 难过
// 情感: negative
// ---
// 文本: 平静
// 情感: neutral
//
// 文本: 兴奋
// 情感:
```

模板与 LCEL 深度结合

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { RunnableSequence } from "@langchain/core/runnables";

// 复杂链：多个模板串联
const analyzeTemplate = ChatPromptTemplate.fromMessages([
  ["system", "你是数据分析师"],
  ["human", "分析以下数据：{data}"]
]);

const reportTemplate = ChatPromptTemplate.fromMessages([
  ["system", "你是报告撰写专家"],
  ["human", "将以下分析结果整理成报告：{analysis}"]
]);

const model = new ChatOpenAI();

// 构建处理链
const chain = RunnableSequence.from([
  // 步骤1：数据分析
  analyzeTemplate.pipe(model),

  // 步骤2：提取内容并传递给下一个模板
  (prevResult) => ({ analysis: prevResult.content }),

  // 步骤3：生成报告
  reportTemplate.pipe(model)
]);

const result = await chain.invoke({
  data: "Q1销售额100万，Q2销售额150万，增长率50%"
});

console.log(result.content);  // 最终报告
```
