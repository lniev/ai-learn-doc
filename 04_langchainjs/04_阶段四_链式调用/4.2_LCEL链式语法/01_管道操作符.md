现代方式：LCEL 管道语法

```javascript
import { ChatOpenAI } from "@langchain/openai";
import { PromptTemplate } from "@langchain/core/prompts";
import { StringOutputParser } from "@langchain/core/output_parsers";

// 1. 创建组件
const prompt = PromptTemplate.fromTemplate("请用一句话介绍{topic}。");
const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });
const outputParser = new StringOutputParser(); // 解析为纯字符串

// 2. 构建链（管道操作符 |）
const chain = prompt.pipe(model).pipe(outputParser);

// 3. 执行（统一使用 invoke）
const result = await chain.invoke({ topic: "区块链" });
console.log(result); // "区块链是一种分布式账本技术..."
```

基础链的完整组成

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";
import { ChatOpenAI } from "@langchain/openai";
import { StringOutputParser } from "@langchain/core/output_parsers";

// 完整基础链：Prompt + Model + Parser
const basicChain = ChatPromptTemplate.fromMessages([
	["system", "你是{role}。"],
	["human", "{question}"],
])
	.pipe(new ChatOpenAI({ temperature: 0.7 }))
	.pipe(new StringOutputParser());

// 执行
const answer = await basicChain.invoke({
	role: "历史学家",
	question: "唐朝有几位皇帝？",
});

console.log(answer); // 纯字符串输出
```

基础链的执行方法

方法 用途 示例
`invoke(input)` 单条执行 `chain.invoke({topic: "AI"})`
`batch(inputs[])` 批量并行 `chain.batch([{topic: "A"}, {topic: "B"}])`
`stream(input)` 流式输出 `for await (const chunk of chain.stream({...}))`

```javascript
// 批量执行
const inputs = [
	{ topic: "机器学习" },
	{ topic: "深度学习" },
	{ topic: "神经网络" },
];

const results = await chain.batch(inputs);
console.log(results); // ["...", "...", "..."]

// 流式执行
const stream = await chain.stream({ topic: "JavaScript" });
for await (const chunk of stream) {
	process.stdout.write(chunk); // 实时输出
}
```

基础链的错误处理

```javascript
import { RunnableSequence } from "@langchain/core/runnables";

// 带错误处理的链
const safeChain = RunnableSequence.from([
	// 步骤1：输入验证
	input => {
		if (!input.topic || input.topic.length > 100) {
			throw new Error("Invalid topic");
		}
		return input;
	},

	// 步骤2：提示模板
	prompt,

	// 步骤3：模型调用（带重试）
	async formattedPrompt => {
		try {
			return await model.invoke(formattedPrompt);
		} catch (error) {
			console.error("Model error:", error);
			return "服务暂时不可用，请稍后重试";
		}
	},

	// 步骤4：输出解析
	outputParser,
]);

const result = await safeChain.invoke({ topic: "AI" });
```

基础链 vs 手动调用的对比

特性 手动调用 基础链（LCEL）
代码量 多步骤分散 一行管道声明
可复用性 低 高
流式支持 需手动实现 原生支持
调试追踪 困难 LangSmith 集成
批量处理 手动循环 内置 `batch()`
