## 11. 场景选择决策树

```
任务复杂度评估
    │
    ├─ 模型预训练已覆盖（常识/简单任务）
    │   └─ Zero-Shot 足够
    │
    ├─ 需要推理但模式标准（数学/逻辑）
    │   ├─ 单步推理 → Zero-Shot CoT
    │   └─ 多步复杂 → Few-Shot CoT（3-5例）
    │
    ├─ 特定格式/风格任务（翻译特定术语、仿写）
    │   └─ Few-Shot（无需CoT，教模式即可）
    │
    ├─ 专业领域+复杂推理（医疗诊断、法律分析）
    │   └─ Few-Shot CoT + 角色设定 + 知识检索
    │
    └─ 需要极高可靠性（关键决策、安全场景）
        └─ 自洽性提示（多条推理链 + 多数表决）
```
