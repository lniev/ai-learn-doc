模板变量高级用法

默认值

```javascript
const template = new PromptTemplate({
  template: "你好{name}，今天是{day|周一}。",  // | 后默认值（部分版本支持）
  inputVariables: ["name", "day"],
  partialVariables: { day: "周一" }  // 编程方式设置默认值
});

// 或运行时 partial
const withDay = await template.partial({ day: "周五" });
const formatted = await withDay.format({ name: "张三" });
```

条件判断（简单逻辑）

```javascript
import { PromptTemplate } from "@langchain/core/prompts";

// 使用函数预处理变量
const template = PromptTemplate.fromTemplate(`
  {greeting}，{name}！
  {statusMsg}
`);

const vars = {
  name: "张三",
  isVip: true,
  greeting: "欢迎",
  statusMsg: (vars) => vars.isVip ? "您是尊贵的VIP用户。" : "普通用户您好。"
};

// 预处理函数变量
const processedVars = {
  ...vars,
  statusMsg: typeof vars.statusMsg === 'function'
    ? vars.statusMsg(vars)
    : vars.statusMsg
};

const result = await template.format(processedVars);
// 输出: 欢迎，张三！您是尊贵的VIP用户。
```
