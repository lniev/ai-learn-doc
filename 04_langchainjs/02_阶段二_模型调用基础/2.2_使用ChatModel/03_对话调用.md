
03 对话调用

核心概念

对话调用 = 多轮消息数组 + 模型调用，保持上下文连贯。

```
单轮：用户输入 → 模型 → 回复
多轮：历史消息 + 新输入 → 模型 → 回复（基于上下文）
```

---

基础多轮对话

```javascript
import { ChatOpenAI, SystemMessage, HumanMessage, AIMessage } from "@langchain/openai";

const model = new ChatOpenAI({ modelName: "gpt-4o-mini" });

// 构建带历史的消息数组
const messages = [
  // 1. 系统设定（可选，但推荐）
  new SystemMessage("你是一位专业的JavaScript导师，用简洁中文回答"),

  // 2. 第一轮对话历史
  new HumanMessage("什么是闭包？"),
  new AIMessage("闭包是函数记住并访问其外部作用域变量的能力，即使外部函数已执行完毕。"),

  // 3. 第二轮对话历史
  new HumanMessage("能举个例子吗？"),
  new AIMessage("function outer() { let count = 0; return () => ++count; }"),

  // 4. 当前问题（基于上文理解）
  new HumanMessage("这个例子中count会被垃圾回收吗？")
];

// 调用（模型能看到完整历史）
const result = await model.invoke(messages);

console.log(result.content);
// 输出：不会，因为返回的内部函数仍引用count，形成闭包...
```

---

简写方式（快速构建）

```javascript
import { ChatPromptTemplate } from "@langchain/core/prompts";

// 使用 fromMessages 简写
const prompt = ChatPromptTemplate.fromMessages([
  ["system", "你是{role}"],
  ["human", "你好，我叫{name}"],
  ["ai", "你好{name}！很高兴认识你。"],
  ["human", "记住我的名字"],
  ["ai", "好的，我记住了你叫{name}。"],
  ["human", "{question}"]  // 当前问题
]);

const chain = prompt.pipe(model);

const result = await chain.invoke({
  role: "助手",
  name: "张三",
  question: "我叫什么？"
});
// 模型回答：你叫张三（基于历史上下文）
```

---

对话调用的三种模式

模式	方法	用途
标准调用	`invoke(messages)`	单次完整响应
批量调用	`batch([messages1, messages2])`	并行处理多个对话
流式调用	`stream(messages)`	实时逐字输出

---

流式对话调用

```javascript
const messages = [
  new SystemMessage("你是诗人"),
  new HumanMessage("写一首关于春天的诗，50字")
];

// 流式输出
const stream = await model.stream(messages);

let fullContent = "";
for await (const chunk of stream) {
  fullContent += chunk.content;
  process.stdout.write(chunk.content); // 实时显示
}

console.log("\n\n完整内容:", fullContent);
console.log("Token消耗:", stream.usage_metadata); // 最后一块包含
```

带 Token 消耗的完整示例

```javascript
async function chatWithHistory(history, newQuestion, modelName = "gpt-4o-mini") {
  const model = new ChatOpenAI({
    modelName,
    temperature: 0.7
  });

  // 构建完整消息数组
  const messages = [
    new SystemMessage("你是 helpful 助手"),
    ...history,  // 展开历史消息
    new HumanMessage(newQuestion)
  ];

  // 调用
  const result = await model.invoke(messages);

  return {
    reply: result.content,
    tokens: result.usage_metadata,  // {input_tokens, output_tokens, total_tokens}
    // 返回更新后的历史（用于下次调用）
    newHistory: [
      ...history,
      new HumanMessage(newQuestion),
      new AIMessage(result.content)
    ]
  };
}

// 使用
let history = [];
let response;

// 第一轮
response = await chatWithHistory(history, "你好，我叫李四");
console.log("AI:", response.reply);
console.log("消耗:", response.tokens);
history = response.newHistory;

// 第二轮（带上下文）
response = await chatWithHistory(history, "我叫什么名字？");
console.log("AI:", response.reply); // "你叫李四"
```

对话历史管理（关键）

```javascript
// 历史过长时的处理策略

// 1. 只保留最近 N 轮
function trimHistory(history, keepRounds = 5) {
  return history.slice(-keepRounds * 2); // 每轮2条消息
}

// 2. 按 Token 数截断
import { encode } from "gpt-tokenizer";

function trimByTokens(history, maxTokens = 3000) {
  let total = 0;
  const trimmed = [];

  // 从后往前加，保留最近的内容
  for (let i = history.length - 1; i >= 0; i--) {
    const tokens = encode(history[i].content).length;
    if (total + tokens > maxTokens) break;
    total += tokens;
    trimmed.unshift(history[i]);
  }

  return trimmed;
}
```
